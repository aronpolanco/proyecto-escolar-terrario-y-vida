# Proyecto escolar Terrario y Vida - Tienda Online

## 📖 Descripción

**Terrario y Vida** es una tienda en línea desarrollada como proyecto escolar. Ofrece productos especializados para la creación y mantenimiento de terrarios, incluyendo plantas, sustratos, decoraciones y accesorios.

##  Productos que ofrecemos

-  **Plantas para terrarios** - Especies ideales para ecosistemas cerrados
-  **Terrarios** - Diferentes tamaños y estilos
-  **Sustratos y piedras** - Base esencial para terrarios saludables
-  **Decoración** - Figuras, piedras decorativas y más
-  **Accesorios** - Herramientas y kits de mantenimiento

##  Funcionalidades

- Catálogo interactivo de productos
- Filtros por categorías
- Carrito de compras dinámico
- Formulario de pedido
- Diseño responsive (adaptable a móviles)
- Interfaz amigable e intuitiva

## Tecnologías utilizadas

- **HTML5** - Estructura de la tienda
- **CSS3** - Estilos, diseño responsive y animaciones
- **JavaScript** - Lógica del carrito, filtros y funcionalidades interactivas
